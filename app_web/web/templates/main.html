О Разделах

Входные данные

Входными данными для работы с формулами является синтаксис LaTeX и изображения. Для наиболее корректной работы с изображениями лучше загружать файлы в формате JPEG. Также, желательно, чтобы на изображении формула имела отступы от краев больше 20px.

Выходные данные

Входными данными является нормализованная формула в формате LaTeX, а также формула в графическом формате KaTeX.

Действия

При работе с формулами вы можете изменять её, чтобы добиться нужного формата. Если формула удовлетворяет Вашим требованиям, её можно сохранить в базу данных.

Работа с БД

На странице отображена полная выгрузка данных из БД. Данные можно скачать в формате Excel.

Загрузка данных

Для пополнения данных в БД множеством формул можно использовать загрузку CSV файла. Формулы добавятся в БД и отобразятся в выгрузке.

Сравнение формул

Для сравнения формул необходимо выбрать формулу из выгрузки и нажать сравнить. В открывшемся окне будет отображена текущая формула и три схожие на неё формулы из БД с процентом схожести.

Что такое LaTeX?

LaTeX – это язык разметки текста, широко используемый в академической среде для создания научных документов, статей и книг. Он особенно популярен при работе с математикой, физикой и другими точными науками благодаря своей способности точно представлять сложные математические выражения.

Для чего используется KaTeX?

KaTeX используется для отображения математических формул непосредственно в браузере. И сходный код формулы выглядит аналогично LaTex, но вместо компиляции он обрабатывается и отображается на веб-странице:
x^{2}+\left(\frac{4y}{5}-\sqrt{|X|}\right)^{2}=1

Почему используются изображения?

Использование изображений для представления математических формул является одним из самых простых способов передачи информации. Данный формат можно использовать практически везде: в презентациях, документах, электронных письмах, социальных сетях и многих других местах. Они не требуют специальной поддержки со стороны программ или платформ, что делает их универсальным решением для передачи математического контента.
